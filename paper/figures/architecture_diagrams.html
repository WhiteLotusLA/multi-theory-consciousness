Figure 1: MTC Framework Architecture

Figure 1. System architecture of the Multi-Theory Consciousness (MTC) framework. Data flows bottom-up: neural substrates generate signals that feed into the consciousness layer, where seven theory modules process them through a four-phase cycle. The resulting ConsciousnessState flows up to the assessment layer, which measures it against 20 indicators on demand. Red dashed arrows indicate cross-theory interactions — data dependencies between modules that create observable interaction effects (Section 3.4). Blue arrows indicate sequential data flow within a single cycle.

Figure 2: Consciousness Cycle Data Flow

Figure 2. Data flow through one consciousness cycle. Candidates enter from the left and are processed through four phases: competition and selection (GWT), self-modeling (AST, HOT), prediction and inference (FEP, BLT), and embodied integration (Damasio). Each module's outputs are listed below its divider line. The red dashed arrow at the top shows the cross-cycle feedback loop: prediction errors from the FEP module modulate candidate salience in the next cycle's workspace competition. IIT and RPT are measured across the full system rather than implemented as sequential cycle steps. The final output is a ConsciousnessState with 30+ fields spanning all seven theories.

Figure 3: Cross-Theory Interaction Map

Figure 3. Cross-theory interaction map. Candidates enter at the top (GWT) and the cycle produces a ConsciousnessState at the bottom. Solid gray arrows show sequential data flow within the consciousness cycle. Red arrows highlight the five cross-theory interactions documented in Section 3.4 — cases where one theory's output substantively affects another theory's computation. Dashed circles (IIT, RPT) indicate theories that are measured across the system rather than implemented as cycle participants. The FEP → GWT salience modulation (left arc) is the strongest interaction: prediction errors from active inference change which candidates win workspace competition in the next cycle, directly linking Friston's framework to Baars'.
